Tutorial Upload Image HD

1. Buka situs Postimages atau Imagekit.io

2. Login menggunakan akun Google Anda.

3. Masuk ke menu Media Library.

4. Klik tombol "New" di kanan atas dan pilih Upload File.

5. Pilih opsi "My Device" lalu unggah foto Anda.

6. Setelah berhasil, Copy URL/Link Langsung dan input di dashboard ini.
